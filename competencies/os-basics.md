# OS Basics

**Category:** Core Technical

## Why it matters

It is often necessary to resolve basic operating system configuration problems before continuing with LT work, as well as to keep your machine in working order.

## Target competency

> Is able to troubleshoot operating system installation issues, configure for optimal use, and teach best practices.

## Progression by component

_Suggested activities to reach each outcome level._

### 1.0 — Can successfully use and troubleshoot current and recent operating systems

| Level | Suggested activities |
| --- | --- |
| **Learner** | Learn to identify the operating system and configure basic options. |
| **Advanced Beginner** | Spend time with an IT professional Installing, activating and configuring an operating system. |
| **Practitioner** | Explore and use the latest appropriate operating system, then help users adjust when they upgrade. |
| **Trainer** | Research and test alternative Operating Systems, like Linux, that may serve some users. |

### 2.0 — Configuration for Success

| Level | Suggested activities |
| --- | --- |
| **Learner** | Identify user needs, such as language, display, accessibility, and preferences |
| **Advanced Beginner** | Learn to configure OS language and display options on your own devices. |
| **Practitioner** | Assist users to configure their devices for display and preferences. |
| **Trainer** | Research assistive technologies for those with disabilities. |
