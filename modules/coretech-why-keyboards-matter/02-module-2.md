# Module 2: Installing Keyman and Keyboards

**Estimated time:** 90 minutes

**Purpose:** Understanding why keyboards matter isn't enough — you need to be able to install Keyman and a keyboard yourself before you can walk anyone else through it.

## Learning objectives

By the end of this module, you will be able to:
- Install the Keyman program from the web
- Install a keyboard for a language you know, either from within Keyman or from keyman.com
- Test a newly installed keyboard in a text editor, confirming it produces the correct characters

## Connect

Before we explain anything, let's try something.

Open Notepad or any word processor. Now try to type a few words in a language you work with that doesn't use the Latin alphabet — just using your normal keyboard.

▶️ *Take 2 minutes and try it now.*

What came out? Probably not what you wanted. That's the problem we're about to fix.

By the end of this module, you'll be able to type correctly in at least one language. That's the goal.

## Content

### Step 1: Install Keyman

First, we install the Keyman engine — the software that makes everything work.

1. Go to [keyman.com/en/windows](https://keyman.com/en/windows/)
2. Click Download Keyman for Windows
3. Open the file when it finishes downloading
4. Windows will ask for permission to install — click Yes. This is normal, even if you're an administrator.
5. Keyman will open automatically. Look for its small icon in the bottom-right corner of your screen (near the clock). That's the system tray.

**TIP** Can't see the icon? Click the small upward arrow (︿) near the clock. Some icons hide there.

Keyman is now installed, but it doesn't have any keyboards yet. Think of it as an empty container — we need to add keyboards next.

### Step 2: Install a keyboard

You can find keyboards two ways.

**From inside Keyman:**
1. Click the Keyman icon in the system tray
2. Open Keyman Configuration
3. Go to Keyboard Layouts and click Download keyboard
4. Search for your language and install

**From the website:**
1. Go to [keyman.com/en/keyboards](https://keyman.com/en/keyboards/)
2. Search for your language
3. Click Install — this opens Keyman and finishes from there

**TIP** You might see several keyboards listed for one language. That's normal — different keyboards suit different purposes. If you're not sure which to pick, look for one marked "recommended" or read the short description.

If someone gives you a `.kmp` file (a Keyman package file), you don't need to search online. Just double-click the file and Keyman will install it for you.

### Step 3: Test it

Let's make sure it works before we go any further.

1. Open Notepad
2. Click the Keyman icon and select your new keyboard from the list
3. Press `Shift + Alt + K` to open the on-screen keyboard pane, so you can see what each key will type
4. Try typing. Compare what appears on screen with the keyboard picture.

**NOTE** `Shift + Alt + K` is one of Keyman's built-in configurable hotkeys (Keyman Configuration → Hotkeys → "Show On Screen Keyboard Pane"), not a generic Windows shortcut — you'll see the full Hotkeys panel in Module 3.

**TIP** Check the system tray icon — it shows which keyboard is active. If you're typing but getting the wrong characters, this is the first place to look.

## Challenge

Now do it yourself, from the beginning.

**✏️** Install Keyman and at least one keyboard for a language you work with. Then open Notepad and type at least one sentence using that keyboard.

**✏️** Take a screenshot of your typed text. Make sure the Keyman icon is visible in the system tray. This is your evidence that it's working.

Don't skip ahead. If something goes wrong during the installation, that's actually useful — make a note of what happened and check Module 4. Real problems are good practice.

## Change

**✏️ Reflection:** What was the hardest part? Was there any step that surprised you or didn't go as expected? Write a short note about it.

This matters because when you help someone else install their first keyboard, you'll remember what tripped you up. That makes you a much better guide.
