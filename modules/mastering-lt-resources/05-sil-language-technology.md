# Mastering the SIL Language Technology Website

**Estimated time:** 45 minutes

**Purpose:** To learn how to use the [SIL Language Technology website](https://software.sil.org/) in LT work.

## Learning objectives

1. You will know how to download software from the SIL Language Technology site.
2. You will know how to find documentation on software from the SIL Language Technology site.

## Connect

**Why should you use the software.sil.org website?**

Software.sil.org is the official software distribution website of **SIL International** (formerly Summer Institute of Linguistics), a nonprofit organization focused on language development, literacy, and Bible translation for minority language communities worldwide.

Here are the key reasons to use it:

- **Trust and legitimacy** — It's the authoritative source for SIL's software tools, so you can be confident you're downloading official, unmodified versions directly from the developers.
- **Specialized linguistic tools** — SIL develops software specifically for language documentation, analysis, and typography that you won't find elsewhere, including tools for working with minority and endangered languages.

Notable software available there includes:

- **FieldWorks Language Explorer (FLEx)** — a comprehensive tool for linguistic fieldwork, lexicography, and language documentation.
- **Paratext** — widely used for Bible translation projects.
- **Keyman** — keyboard input software for typing in hundreds of scripts and languages.
- **Bloom** — for creating and sharing books in local languages.
- **Font collections** — high-quality Unicode fonts covering scripts that commercial fonts often lack (e.g., Charis SIL, Gentium, Doulos SIL).

Other reasons to use the site:

- **Free and open** — Most SIL software is free, and much of it is open source, making it accessible to communities and researchers with limited resources.
- **Support for rare scripts** — If you work with non-Latin writing systems, complex scripts, or languages with special diacritics, SIL's fonts and keyboard tools are often the best or only option.
- **Active development and support** — SIL maintains documentation, forums, and ongoing updates for their tools.

It's particularly valuable for **linguists, translators, missionaries, language activists, and educators** working with minority or indigenous languages.

## Content

The SIL software website serves as a centralized hub for accessing a wide array of language technology tools and typography resources. Users can navigate the platform to find active software products, experimental web applications, and older programs that are no longer officially supported. Individual product pages are structured to provide installation files, comprehensive educational tutorials, and the latest development news. Additionally, the site features a dedicated section for global and regional fonts, offering specialized character sets for diverse writing systems. By utilizing this resource, individuals can gain the technical documentation and files necessary to support linguistic projects and script development worldwide.

**Question:** Your organization needs to find a reliable SIL font for a complex regional writing system, download the latest version, and check which characters it supports. Using the SIL software site, walk through where you would go and what you should keep in mind about the character set.

- Since the need is for a **complex regional writing system**, you would look for a **regional/script-specific font** rather than a global font, as these are highly specialized for that purpose. Once you find the right font, navigate to the **Downloads** section of its product page, which provides both current and legacy versions. To check which characters it supports, look for the **specific character set** listed under its resources, along with installation instructions.

**Question:** A colleague tells you she found a useful-looking tool on the SIL site, but it is listed under the unsupported software category. She is hesitant to use it and asks for your advice. What would you tell her about what "unsupported" actually means, and what should she watch out for?

- You would tell her that **unsupported software consists of older tools that no longer receive active maintenance** from SIL. This does not mean she is forbidden from using it — she is completely permitted to do so, and these tools can still be very useful in specific edge cases. However, the key thing to watch out for is that **she should not expect active technical support from SIL if she runs into a bug or problem**. It is an important distinction: the tool may still work well for her needs, but she will essentially be on her own if something goes wrong.

**Question:** You are preparing to download a SIL software tool for the first time. Walking through the product subsite, what are the four key sections you should check in order, and what specific information does each one give you?

- **Step 1 — Features:** Check this first to confirm the tool actually does what you need before investing any more time. This prevents you from downloading something that turns out to be the wrong fit for your work.
- **Step 2 — Downloads:** Once you have confirmed the tool is right for you, head here to find the installer for your operating system.
- **Step 3 — Learn:** This is where you find the full documentation. It goes far beyond a basic FAQ — for a tool like Scripture App Builder, this page includes step-by-step installation instructions, comprehensive building guides, publishing checklists, links to complementary tools like Audacity and HearThis, video tutorials, and even tips for running your own training workshops.
- **Step 4 — News:** Check this tab last to stay current on the latest release announcements, feature updates, and development roadmaps.
