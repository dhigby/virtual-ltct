# CoreTech: Why Keyboards Matter

A short course on why default keyboards fall short for language technology work, and how
Keyman solves the problem. Developed and drafted in Cypher for Business, but not yet
delivered to learners there or anywhere else; this repo's content was revised from a
Claude.ai-chat draft with every specific technical claim verified live against Windows 11 +
Keyman by the SME — see `00-design.md` for the full verification record.
