# Scenario bank

**Purpose:** Applied practice scenarios learners work through after the lesson content,
before the quiz. **Format:** sequence from foundational → intermediate → complex, matching
the pattern in `modules/coretech-computer-hardware/04-scenario-bank.md`.

## How to use these scenarios

### For learners
- Read each scenario carefully.
- Apply the diagnostic/decision framework taught in the lesson content.
- Write a detailed response.
- Submit to your mentor for feedback.

### For mentors
- Use the accompanying mentor guide for assessment criteria.
- Look for systematic thinking, not just a "correct" answer.
- Provide constructive feedback.
