## Suggested Enhancements

Since this is a software walkthrough, users benefit from knowing exactly where to look.
### 2. Clarifying the "Renderings vs. Text" Distinction
In **Step 4**, it is vital to emphasize the difference between **recording** a rendering in the tool and **updating** the actual Scripture text. Experienced users often assume the tool is a "global replace" feature.
> **💡 Pro-Tip:** Adding a rendering to the list tells Paratext what to *look* for. It does not automatically change the words in your verses. After adding a rendering, use the **Renderings** pane to click the **blue arrow** or use **Ctrl+R** to apply the rendering to the verse.

- **Recommendation:** Add a note about **Denying** renderings. If the tool flags "Peter" in a verse where the text intentionally says "Cephas," the translator needs to know how to "Deny" that occurrence so the Red X disappears without changing the text.
