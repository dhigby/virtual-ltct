### Content (15 minutes)
Watch the video Language Dev Workflow.
Language development follows a systematic process. Understanding this workflow helps you recommend the right tools at the right time.
**The Basic Workflow:**
1. **Word Collection** → Building vocabulary with meanings and grammar notes
2. **Phonological Analysis** → Understanding the sound system of the language
3. **Orthography Development** → Creating a writing system (this is iterative—you’ll refine it based on phonology)
4. **Parallel Work Begins:**
	- Linguistics (lexicon development and grammar description)
	- Translation (Scripture and other texts)
	- Literacy (teaching materials)
**Key Insight:** Orthography development is the foundation. Once established, linguistics, translation, and literacy work can all happen at the same time.
**Two Critical Outputs:**
- **Phonology statement** - Written for outsiders (linguists, academics)
- **Orthography statement** - Written for the language community
