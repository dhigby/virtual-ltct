## Assessment Quiz (22 questions)

This quiz covers all three modules of CoreTech: OS Basics — OS Fundamentals and
Identification, Installation and Configuration, and Troubleshooting and Maintenance.
You need **80% (18/22)** to pass.

### Section 1: What an OS Does & the Three-Jobs Framework (Questions 1-3)

- A) Managing hardware resources, running and managing applications, providing the user interface
- B) Installing software, connecting to the internet, and running antivirus scans
- C) Managing hardware resources, managing user accounts, and rendering fonts
- D) Booting the computer, displaying the desktop, and printing documents

**Question 2:** A user tells you: "I can't find my files, my settings won't save, and I'm having keyboard input problems." Which of the OS's three jobs does this point to?
- A) Managing hardware resources
- B) Running and managing applications
- C) Providing the user interface

- B) False
