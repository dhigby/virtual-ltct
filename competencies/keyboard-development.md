# Keyboard Development

**Category:** Technology Domain

## Why it matters

If keyboarding solutions are non-existent or poorly adapted to input of local languages, a Language Technologist may be needed to develop a new solution,

## Target competency

> Can develop and trains others to develop custom keyboard and data-entry tools.

## Progression by component

_Suggested activities to reach each outcome level._

### 1.0 — Typing Keyboard Development

| Level | Suggested activities |
| --- | --- |
| **Learner** | Examine local and nearby keyboards to learn the strategies used for text input. |
| **Advanced Beginner** | Learn the keyboard coding environment and how to compile a typing keyboard |
| **Practitioner** | Create and distribute a custom typing keyboard solution |
| **Trainer/Proficient** | Work with individuals to create custom typing keyboarding solutions. Submit keyboards to appropriate repositories for web distribution. |
| **Expert** | Teach a group how to create custom keyboard solutions for redistribution. Create and submit OS-native keyboards to appropriate repositories. |

### 2.0 — Touch Keyboard Development

| Level | Suggested activities |
| --- | --- |
| **Learner** | Examine local and nearby keyboards to learn the strategies used for text input on mobile devices. |
| **Advanced Beginner** | Learn the keyboard coding environment and how to compile a touch keyboard |
| **Practitioner** | Create and distribute a custom touch keyboard solution |
| **Trainer/Proficient** | Work with individuals to create custom touch keyboarding solutions. Submit keyboards to appropriate repositories for web distribution. |
| **Expert** | Teach a group how to create custom touch keyboard solutions for redistribution. |

### 3.0 — Language Model Development

| Level | Suggested activities |
| --- | --- |
| **Learner** | Learn how to create ordered word lists with frequency counts |
| **Advanced Beginner** | Collect texts and use a tool to create a language model |
| **Practitioner** | Create a predictive keyboard by adding a language model to an existing keyboard. |
| **Trainer/Proficient** | Work with individuals to create language models for text entry. Submit language models to appropriate repositories for web distribution. |
| **Expert** | Teach a group how to create language models for redistribution. Explore configuration of morphological options in language models. |
