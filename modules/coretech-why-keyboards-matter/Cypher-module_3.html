Module 3: Configuring System Settings

In the last module, you installed the Keyman program and a keyboard for a language.  In this session, you will learn how to configure the Operating System settings so that the computer will automatically change to the right keyboard when you set the language.

“My keyboard is installed—but Windows keeps switching to the wrong one!”

Something wrong with the configuration?

  • Key concepts:

    • Windows languages and Keyman keyboards are separate things

    • You can link a Keyman keyboard to a language in Windows

    Step-by-step:

    1. Open Keyman Configuration

    2. Choose a Keyman keyboard

    3. Click "Associated Language" and pick a matching Windows language (or create one)

    4. Set it as default for that language

    5. Use hotkeys to switch between languages/keyboards:

      • Windows: Alt+Shift or Win+Space

      • Keyman menu for direct switching

  • Switch methods: hotkeys, language bar, Keyman menu
