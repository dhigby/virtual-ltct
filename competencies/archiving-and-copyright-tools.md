# Archiving and Copyright Tools

**Category:** Technology Domain · **Workbook label:** Archiving & Copyright

## Why it matters

Perservation of our data is important. It prevents data loss, meets legal requirements and increases secuity.

## Target competency

> Can promote archiving and proper use and licensing of intellectual property.

## Progression by component

_Suggested activities to reach each outcome level._

### 1.0 — General

| Level | Suggested activities |
| --- | --- |
| **Learner** | Identify the various Creative Commons licenses. Learn what metadata is. Complete the SIL intellectual property courses. |
| **Advanced Beginner** | Learn how to use the licensing components properly in various LT tools. |
| **Practitioner** | Learn how to use REAP and RAMP for archiving data |
| **Trainer/Proficient** | Teach others to use the licensing components in LT tools |
| **Expert** | Teach others to use REAP and RAMP |
