# Module 6: Grammar Description & Documentation

### Connect (5 minutes)
**Reflection:** Language work is more than just translation. Grammar descriptions are essential for training, reference, and preserving linguistic knowledge for future generations.
Think about: How do grammar tools help linguists document language structure systematically?
### Content (25 minutes)
Play the video Grammar Description Tools.
**Grammar Description Tools**
**FLEx (Discourse Analysis & Grammar)**
- Text charting feature for discourse analysis
- Understands linguistic charting needs
- Produces grammar sketches from morphological analysis
- Exports to HTML
**PAWS (Parser and Writer for Syntax)** - Expert system for grammar writing
- Easy-to-use interface
- Produces \~100-page rough draft of syntax grammar
- Based on answers to guided questions
- Output loaded into XLingPaper for final editing
**LingTree** - Linguistic tree diagrams
- Creates syntactic, morphological, and phonological tree diagrams
- Outputs PNG and SVG formats
- Easy graphic visualization of linguistic structures
**XLingPaper** - Linguistic document publishing
- Authors, archives, and publishes linguistic research
- Single source → multiple outputs (web pages, PDFs, Word docs, Open Office, ePUB)
- Automates numbering, citations, glosses, abbreviations
- Lets linguists focus on content, not formatting
**Purpose:** Understanding natural ways of expression and documenting grammar for reference and training
### Challenge (20 minutes)
**Tool Matching:**
Match each need with the appropriate tool:
1. A linguist wants to publish a 50-page grammar description with proper linguistic formatting
2. A team needs to create syntactic tree diagrams for training materials
3. A linguist wants to generate a rough draft of syntax grammar through guided questions
4. A project needs to analyze discourse structure in narrative texts
**Tools:** XLingPaper, LingTree, PAWS, FLEx
**Case Study:**
A linguist has been collecting grammatical data for 3 years in notebooks and Word documents. They need to produce a comprehensive grammar description for publication.
**Task:** Recommend a workflow using the grammar tools. Which tool would you use for:
- Generating initial grammar draft?
- Creating tree diagrams?
- Final publication formatting?
