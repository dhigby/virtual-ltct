# Module 4: Literacy Primers & Materials Development

### Connect (5 minutes)
**Consider this:** A community receives Scripture in their language—but most people can’t read it. Literacy work and Scripture translation must go hand-in-hand.
Have you seen situations where translation outpaced literacy development? Or vice versa? What happened?
### Content (30 minutes)
### Primer Development Tools
Play the video Literacy Tools
**Primer Purpose:** Teach people to read their own language systematically by introducing letters/sounds in a carefully designed order.
**PrimerPro** - Comprehensive primer creation
- Produces tentative teaching order for graphemes (letters, digraphs, consonant clusters)
- Makes controlled vocabulary story writing more pleasant and creative
- Uses only graphemes taught in previous lessons
- Provides usable word lists for each lesson
- Checks stories automatically
- Does tedious work that authors traditionally did manually
- Authors retain creativity while getting systematic support
**PrimerPrep** - Simplified primer tool
- Similar to PrimerPro
- Easier to use but less powerful
- Good for teams wanting simpler workflow
**Publishing workflow:** Primer tool (PrimerPro/PrimerPrep) → Desktop publishing (MS Publisher) → Final primer
**Output:** Basic primer teaching reading systematically
### Literacy Materials Creation
**Bloom** - Primary literacy tool
- Creates simple books easily
- Translates books into multiple languages
- Online crowd-sourced library (download and translate existing books)
- Produces PDFs, ePubs, audiobooks
- Talking books with text highlighting
- Highlights text while reading aloud (facilitates learning to read)
**Bloom Reader** - Mobile reading app
- Smartphone access to Bloom books
- Highlights text while playing audio
- Facilitates learning to read
- Allows communities to download Bloom talking books
- Text highlighting synchronized with audio
- Supports literacy learning
**Reading App Builder** - Story collections
- Creates Android reading apps
- Makes literacy materials accessible on phones
- Creates apps with story collections
- Mobile-friendly reading materials
**Alpha Tiles** - Literacy games
- Creates basic literacy games for new languages
- Can be developed in less than one week
- Features continually added
**Traditional desktop publishing**
- MS Publisher or similar tools
- Used for primers and final literacy materials
**Output:** Comprehensive literacy materials in vernacular language (books, apps, games)
### Challenge (20 minutes)
**Literacy Program Planning:**
A language community has:
- New orthography (just finalized)
- 200-word lexicon
- No existing literacy materials
- Mixed oral and literate preferences
- Most adults have smartphones
**Task:** Design a 12-month literacy development plan. Answer:
1. What tool would you use FIRST to develop the primer?
2. How would you create supplemental reading materials after the primer?
3. How would you make materials accessible on smartphones?
4. What’s the relationship between primer development and lexicon building?
**Troubleshooting:**
A literacy worker emails: “I used PrimerPro to create a word list for lesson 5, but two of the words contain a digraph we haven’t taught yet. Why is PrimerPro suggesting these words?”
What’s likely happening? How does PrimerPro’s controlled vocabulary feature work?
**Literacy Scenario:**
A literacy worker asks: “I want to create 20 simple books for new readers, with audio so they can hear and read along. I also want these on phones. What’s the most efficient workflow?”
What single tool handles most of these requirements?
### Change (10 minutes)
**Reflection:**
- Why must primer development follow orthography development?
- How do primers differ from other reading materials?
- What’s the advantage of creating literacy materials in multiple formats (print, apps, audiobooks)?
- How do literacy and translation efforts support each other?
- Can you explain why orthography development must happen before serious translation and literacy work?
