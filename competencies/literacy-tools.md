# Literacy Tools

**Category:** Technology Domain

## Why it matters

This competency covers the usage and training of users to use the current literacy and education tools to produce primers and easy readers..

## Target competency

> Is able to use the tools that assist literacy and education workers in their work and is able to consult others in their use. This includes primer development and easy reader development.

## Progression by component

_Suggested activities to reach each outcome level._

### 1.0 — Literacy Tools

| Level | Suggested activities |
| --- | --- |
| **Learner** | Demonstrates basic knowledge of literacy tools |
| **Advanced Beginner** | Can use literacy tools and troubleshoot issues that arise |
| **Practitioner** | Can produce literacy materials using these tools. |
| **Trainer/Proficient** | Prepare and teach a literacy tools training event. |
| **Expert** | Train someone to teach literacy tools |

### 2.0 — Creating Simple Books

| Level | Suggested activities |
| --- | --- |
| **Learner** | Learn to create and translate simple books with illustrations using recommended software. |
| **Advanced Beginner** | Learn advanced formatting and customization of simple books with illustrations |
| **Practitioner** | Help individuals to write, translate, and publish simple books through Literacy software. |
| **Trainer/Proficient** | Train a group to write, translate, and publish simple books in Literacy Software |
| **Expert** | Train trainers to teach users to write, translate, and publish simple books through Literacy Software. |

### 3.0 — Creating primers, decodable or leveled readers

| Level | Suggested activities |
| --- | --- |
| **Learner** | Learn the tools used to create primers, decodable and leveled readers |
| **Advanced Beginner** | Learn to create primers, decodable readers, leveled readers, and comics using recommended software. |
| **Practitioner** | Learn to determine teaching order of graphemes and to advise others on the appropriate order for literacy publications. |
| **Trainer/Proficient** | Help individuals to create primers, decodable readers, leveled readers, and comics through Literacy software. |
| **Expert** | Train a group to create primers, decodable readers, leveled readers, and comics in Literacy Software. Help a group to use tools to determine a teaching order for letters |

### 4.0 — Interactive Literacy Publications

| Level | Suggested activities |
| --- | --- |
| **Learner** | Learn available tools that make literacy interactive publication |
| **Advanced Beginner** | Create an audio book with highlighted text |
| **Practitioner** | Create an interactive literacy publication |
| **Trainer/Proficient** | Train someone to use available tools for making interactive literacy publications |
| **Expert** | Train someone to teach the use of available tools to make interactive literacy publications |
