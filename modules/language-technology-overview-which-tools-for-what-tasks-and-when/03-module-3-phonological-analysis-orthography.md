# Module 3: Phonological Analysis & Orthography

### Connect (5 minutes)
Say these words out loud: “rough,” “though,” “through,” “cough.”
Notice how “-ough” makes different sounds? English spelling is inconsistent. When developing an orthography for a new language, you get to make better choices—but first you need to understand the sound system.
### Content (15 minutes)
Play the video Phonological Analysis Tools.
**Phonological Analysis Tools**
Phonological analysis answers: “What sounds exist in this language, and how do they work together?”
**Phonology Assistant** - Environment-based analysis
- Shows sounds and where they occur in words
- Supports IPA (International Phonetic Alphabet)
- Helps identify patterns and contrasts
**PTEST (Phonology Template Editor and Search Tool)** - Flexible analysis
- Supports both IPA and intuitive writing systems
- Template-based searching for sound patterns
**Dekereke** - Comprehensive phonological analysis aid
- Maintains and processes phonetic/phonological data
- Search and analysis functions
- Answers questions about phonological patterns
**Speech Analyzer** - Audio visualization
- Plays audio files
- Displays sound waves
- Determines actual sounds in recorded words
- Useful for comparing similar sounds
**Output:** A phonology statement describing the sound system
**The Iterative Process**
1. Collect words → Analyze sounds → Develop trial orthography
2. Test trial orthography → Refine phonological analysis → Improve orthography
3. Repeat until you have a stable, community-accepted writing system
**Why this matters:** A poorly designed orthography creates problems for decades. People struggle to read, literacy materials need constant revision, and the community may reject it.
### Challenge (10 minutes)
**Case Study:**
A language has two sounds that are similar: \[t\] and \[th\] (like in English “top” and “thin”).
The phonological analysis shows:
- \[t\] appears at the beginning, middle, and end of words
- \[th\] only appears at the beginning of words, before the vowels \[i\] and \[e\]
**Questions:**
1. Are these two separate sounds (phonemes) or variations of one sound (allophones)?
2. How many letters should the orthography use for these sounds?
3. What tool would help you visualize where each sound occurs?
**Scenario:** A team contacts you: “We’re using Phonology Assistant but we’re confused. Do we need to know IPA to use it effectively?”
How do you respond?
