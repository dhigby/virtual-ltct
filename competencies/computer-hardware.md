# Computer Hardware

**Category:** Core Technical

## Why it matters

While hardware support is not a goal of a Language Technologist, in the absence of a dedicated technician, it is often necessary to resolve basic hardware problems before continuing with LT work. This competency invites the candidate to stay up-to-date with the use and troubleshooting of current and coming hardware and its basic usage. Those hoping to specialize in Language Technology will need to strike a healthy balance between LT and IT to progress in LT.

## Target competency

> Is able to troubleshoot basic computer hardware problems.

## Progression by component

_Suggested activities to reach each outcome level._

### 1.0 — Can effectively use and troubleshoot current and recent computer and mobile hardware

| Level | Suggested activities |
| --- | --- |
| **Learner** | Identify the various hardware components of a computer and understand their function. |
| **Advanced Beginner** | Spend time with an IT professional while they troubleshoot computer hardware issues. |
| **Practitioner** | Troubleshoot hardware issues, and recommend appropriate hardware for language program use. |
| **Trainer** | Research the latest hardware components |
